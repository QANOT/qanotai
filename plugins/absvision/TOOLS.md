# AbsVision HR ‚Äî Mavjud toollar

## Xodimlar
- `absvision_get_employees` ‚Äî Xodimlar ro'yxati (filter: bo'lim, holat, qidiruv)
- `absvision_get_employee` ‚Äî Bitta xodim tafsilotlari
- `absvision_get_employee_options` ‚Äî Xodimlar dropdown (ID va ism)
- `absvision_employee_career` ‚Äî Xodim karyera tarixi
- `absvision_employee_documents` ‚Äî Xodim hujjatlari

## Bo'limlar va Lavozimlar
- `absvision_get_departments` ‚Äî Bo'limlar ro'yxati
- `absvision_get_positions` ‚Äî Lavozimlar ro'yxati
- `absvision_get_branches` ‚Äî Filiallar ro'yxati

## Davomat
- `absvision_attendance_dashboard` ‚Äî Boshqaruv paneli (bugun kim keldi/kelmadi/kechikdi)
- `absvision_attendance_logs` ‚Äî Kirish-chiqish loglari
- `absvision_attendance_summaries` ‚Äî Kunlik davomat xulosalari
- `absvision_attendance_team_summary` ‚Äî Jamoa umumiy statistikasi
- `absvision_attendance_inout_stats` ‚Äî Kirish-chiqish statistikasi
- `absvision_attendance_audit` ‚Äî Shubhali yozuvlar audit
- `absvision_attendance_monthly` ‚Äî Oylik statistika

## Ta'tillar
- `absvision_leave_types` ‚Äî Ta'til turlari
- `absvision_leave_requests` ‚Äî Ta'til so'rovlari
- `absvision_leave_request_detail` ‚Äî So'rov tafsilotlari
- `absvision_leave_balances` ‚Äî Ta'til qoldiqlari
- `absvision_holidays` ‚Äî Bayramlar ro'yxati

## Oshxona
- `absvision_canteen_menu` ‚Äî Bugungi menyu
- `absvision_canteen_transactions` ‚Äî Ovqatlanish tarixi
- `absvision_canteen_monthly` ‚Äî Oylik xarajatlar

## Profil
- `absvision_my_dashboard` ‚Äî Shaxsiy dashboard
- `absvision_my_profile` ‚Äî Profil ma'lumotlari
- `absvision_attendance_today` ‚Äî Bugungi davomat holati

## Eksport
- `absvision_export_attendance` ‚Äî Davomat XLSX/CSV eksport (yuklab olish havolasi)
- `absvision_export_employees` ‚Äî Xodimlar XLSX/CSV eksport (yuklab olish havolasi)
